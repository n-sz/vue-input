# 概要

vue.jsの練習用リポジトリです

# vue.js install

## 1. vue.jsのインストール

```
$ npm install vue
+ vue@2.5.22
added 1 package from 1 contributor and audited 1 package in 1.986s
found 0 vulnerabilities
```

## 2. vue-cliのインストール

```
$ npm install --global vue-cli
+ vue-cli@2.9.6
added 239 packages from 206 contributors in 24.786s
```
